curl 기반 파일 공유 서비스

Pastebox

curl을 사용하여 파일이나 텍스트를 업로드하면 5자리의 무작위 URL이 생성됩니다. 생성된 링크는 30일 후에 자동으로 삭제됩니다.

텍스트 업로드

echo "hello" | curl -X POST --data-binary @- {{ .BaseURL }}/

파일 업로드

curl -F "file=@test.txt" {{ .BaseURL }}/

비밀번호 보호 링크

curl -H "usepassword: true" -F "file=@secret.txt" {{ .BaseURL }}/

영구 보관

curl -H "data-policy: permanent" -F "file=@server-logs.log" {{ .BaseURL }}/

만료 정보

임시 업로드의 경우 업로드 응답에 만료일이 포함됩니다. 영구 업로드의 경우 만료일이 포함되지 않습니다.

주소: {{ .BaseURL }}/AbC12
만료일: 2026-06-24T05:10:26Z
삭제링크: {{ .BaseURL }}/AbC12?delete=DELETE_TOKEN

수동 삭제

업로드할 때마다 삭제 URL이 반환됩니다. 해당 URL에 접속하거나 curl을 통해 요청을 보내면 업로드된 콘텐츠를 수동으로 삭제할 수 있습니다.

curl "{{ .BaseURL }}/AbC12?delete=DELETE_TOKEN"
